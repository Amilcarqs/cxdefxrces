# PROBLEMA
Two soldiers are playing a game. At the beginning first of them chooses a positive integer `n` and gives it to the second soldier. Then the second one tries to make maximum possible number of rounds.

Each round consists of:

* choosing a positive integer `x > 1`
* such that `n` is divisible by `x`
* replacing `n` with `n / x`

When `n` becomes equal to `1` and there are no more valid moves, the game ends.

The score of the second soldier is equal to the number of rounds he performed.

To make the game more interesting, the first soldier chooses `n` of the form:

```text id="ulr4e2"
a! / b!
```

for some positive integers `a` and `b` (`a >= b`).

Here:

```text id="cf1c1h"
k! = 1 * 2 * 3 * ... * k
```

---

##· Input
The first line contains an integer:

```text id="2zmk8l"
t
```

the number of games.

Then follow `t` lines.

Each line contains two integers:

```text id="mb4t3s"
a b
```

Constraints:

```text id="9p1mbv"
1 <= t <= 1,000,000
1 <= b <= a <= 5,000,000
```

---

## Output

For each game output:

> the maximum possible score of the second soldier.

---

## Example

Input:

```text id="vuhk4n"
2
3 1
6 3
```

Output:

```text id="jvbxv5"
2
5
```

---

## Explanation

## Case 1

```text id="v3c7q8"
3! / 1!
=
6
=
2 * 3
```

Two prime factors → answer = 2.

---

# Explicacion completa de la solucion

## 1. Que pide el problema realmente

El juego consiste en dividir el numero la mayor cantidad de veces posible.

Cada vez conviene dividir por un numero primo, porque asi haces mas movimientos.

Ejemplo:

12 = 2 * 2 * 3

Puedes hacer:

* dividir entre 2
* dividir entre 2
* dividir entre 3

Total: 3 movimientos.

Entonces el problema realmente pide:

> contar cuantos factores primos tiene el numero.

Contando repeticiones.

---

# 2. El numero del problema

El numero es:

a! / b!

Ejemplo:

6! / 3!

= (1 * 2 * 3 * 4 * 5 * 6)/(1 * 2 * 3)

= 4 * 5 * 6

Entonces no necesitas calcular factoriales enormes.

Solo necesitas contar factores primos desde:

b+1 hasta a.

---

# 3. Idea general de la solucion

Queremos:

1. Saber cuantos factores primos tiene cada numero.
2. Responder consultas rapidamente.

Para eso usamos:

* SPF
* prefijos

---

# 4. Que es SPF

SPF significa:

Smallest Prime Factor

o sea:

> el factor primo mas pequeño de un numero.

---

# 6. Para que sirve SPF

Sirve para factorizar numeros rapidamente.

Ejemplo con 12:

spf[12] = 2

Entonces:

12 = 2 * 6

Ahora:

spf[6] = 2

Entonces:

6 = 2 * 3

Ahora:

spf[3] = 3

Entonces:

3 = 3 * 1

Resultado:

12 = 2 * 2 * 3

Tiene 3 factores primos.

---

# 7. Construccion de la criba SPF

Codigo:

```cpp
vector<int> smallest_prime_factor(LIMITE + 1, 0);

for (int posible_primo = 2; posible_primo <= LIMITE; posible_primo++)
{
    if (smallest_prime_factor[posible_primo] == 0)
    {
        for (int multiplo = posible_primo;
             multiplo <= LIMITE;
             multiplo += posible_primo)
        {
            if (smallest_prime_factor[multiplo] == 0)
            {
                smallest_prime_factor[multiplo] = posible_primo;
            }
        }
    }
}
```

# 13. Calcular cantidad de factores primos

Codigo:

```cpp
for (int numero = 2; numero <= LIMITE; numero++)
{
    cantidad_factores_primos[numero] =
        cantidad_factores_primos[
            numero / smallest_prime_factor[numero]
        ] + 1;
}
```

---

# 14. Como funciona esta formula

La idea es:

> quitar un factor primo cada vez.

Ejemplo con 12:

spf[12] = 2

Entonces:

# cantidad[12]

cantidad[6] + 1

Ahora:

# cantidad[6]

cantidad[3] + 1

Ahora:

# cantidad[3]

cantidad[1] + 1

Y:

cantidad[1] = 0

Entonces:

cantidad[3] = 1

cantidad[6] = 2

cantidad[12] = 3

# 16. Prefijos

Codigo:

```cpp
for (int numero = 1; numero <= LIMITE; numero++)
{
    prefijo_factores[numero] =
        prefijo_factores[numero - 1]
        + cantidad_factores_primos[numero];
}
```

---

# 17. Que guarda prefijo

Guarda la suma acumulada.

Ejemplo:

| numero | factores | prefijo |
| ------ | -------- | ------- |
| 1      | 0        | 0       |
| 2      | 1        | 1       |
| 3      | 1        | 2       |
| 4      | 2        | 4       |
| 5      | 1        | 5       |
| 6      | 2        | 7       |

---

# 18. Como responder queries

Codigo:

```cpp
cout << prefijo_factores[a] - prefijo_factores[b];
```

---

# 19. Por que funciona

Porque:

a! / b!

equivale a multiplicar:

(b+1)*(b+2)*...*a

Entonces:

la cantidad total de factores primos es:

suma hasta a
menos
suma hasta b

Y eso es exactamente:

```cpp
prefijo[a] - prefijo[b]
```
